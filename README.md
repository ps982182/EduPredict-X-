# 🎓 EduPredict-X+

EduPredict-X+ is an AI-powered system that predicts student performance risk, explains the reasons, and suggests improvements.

---

## 🚀 Features

- 📊 Risk Prediction (XGBoost)
- 🧠 Explainable AI (SHAP)
- 🎯 Actionable Suggestions
- 💻 Interactive Dashboard (React)

---

## 🏗️ Tech Stack

- Frontend: React, Recharts, Framer Motion  
- Backend: FastAPI  
- ML: XGBoost, Scikit-learn, SHAP
